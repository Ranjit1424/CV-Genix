# CV-Genix

CV-Genix is an AI-powered resume builder that helps users create, improve, and tailor resumes for specific job roles. It includes a React frontend and a Spring Boot backend with AI-based resume generation and analysis features.

## Features

- AI resume generation
- Resume tailoring based on job descriptions
- ATS-style resume rating and feedback
- Multiple resume templates and preview options
- PDF export
- Saved resume history
- Interview question generation
- Job search integration
- User authentication and profile support

## Tech Stack

- Frontend: React, Vite, Tailwind CSS
- Backend: Spring Boot, Java
- Database: MySQL
- AI Integration: Ollama
- Other Tools: Maven, Axios

## Project Structure

- `Resume_frontend` - Frontend application
- `Create-Resume` - Backend application

## Prerequisites

Before running the project, make sure you have installed:

- Node.js
- Java 21
- Maven
- MySQL
- Ollama
